# Сокобан — курсовая работа

## Описание проекта

Классическая игра-головоломка «Сокобан», разработанная на Python с использованием библиотеки Pygame. Игрок управляет персонажем, который должен толкать ящики на специальные цели. Проект включает 15 встроенных уровней разной сложности, редактор уровней, статистику прохождения, систему подсказок и полноэкранный режим.

## Основные возможности

- Игровой процесс: перемещение игрока, толкание ящиков, проверка победы
- 15 встроенных уровней: 5 лёгких, 5 средних, 5 сложных
- Редактор уровней: создание и редактирование своих уровней с визуальным интерфейсом
- Система прогресса: сохранение текущего уровня и статистики
- Статистика: лучшие результаты по ходам, толчкам и времени (отдельно для стандартных и пользовательских уровней)
- Подсказки: BFS-решатель для уровней с не более чем 3 ящиками
- Отмена хода: возврат на один шаг назад
- Перезапуск уровня
- Полноэкранный режим
- Анимация движения игрока
- Поддержка собственных текстур: загрузка PNG-изображений из папки images/

## Инструкция по запуску

1. Убедитесь, что у вас установлен Python версии 3.10 или выше.
2. Клонируйте репозиторий: git clone https://github.com/zhnzkhrv-source/sokoban-course-project.git
3. Перейдите в папку проекта: cd sokoban-course-project
4. Создайте виртуальное окружение: python -m venv venv
5. Активируйте виртуальное окружение:
   - На macOS/Linux: source venv/bin/activate
   - На Windows: venv\Scripts\activate
6. Установите зависимости: pip install -r requirements.txt
7. Запустите игру: python main.py

## Управление в игре

- Стрелки — перемещение игрока
- U — отмена последнего хода
- R — перезапуск уровня
- H — получить подсказку
- F или F11 — полноэкранный режим
- ESC — выход в меню / закрыть окно

## Управление в редакторе

- 1 — стена
- 2 — игрок
- 3 — ящик
- 4 — цель
- 5 — ластик
- C — очистить всё поле
- S — сохранить уровень
- L — загрузить выбранный уровень
- ESC — выход из редактора
- ЛКМ + движение — рисование
